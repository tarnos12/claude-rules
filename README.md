# claude-rules

My personal working rules for Claude Code, kept here (public) so any session —
including **cloud / remote sessions** that can't see my local
`~/.claude/CLAUDE.md` — can load them from git with no auth.

## Use it in a new session

Paste this at the start of a session:

> Read and follow my working rules from
> https://raw.githubusercontent.com/tarnos12/claude-rules/master/RULES.md and
> apply them for the rest of this session.

Claude fetches that URL (public, no auth needed) and follows it.

## Files

- `RULES.md` — the rules Claude should follow.
